部署教程
从零开始搭建团队 30 分钟完成
01
安装 OpenClaw
在 Mac/Linux 安装运行底座
npm install -g openclaw
openclaw init && openclaw gateway start
02
创建 Discord Bot
在 Discord Developer Portal 为每个 Agent 创建独立 Bot，获取 Token，邀请进服务器（需开启 Message Content Intent）
03
配置 openclaw.json
主管设 requireMention: false，专员设 true。主管用 Opus 模型，专员用 Sonnet
04
创建工作空间 + SOUL.md
每个 Agent 独立目录，SOUL.md 定义角色职责、响应规则、工具偏好
05
安装 Skills
openclaw skills install listing-writer
openclaw skills install rufus-geo-optimizer
openclaw skills install linkfoxagent
06
存活测试验证
在频道发 @主管 测试所有 agent 是否正常工作，5个Agent 40秒内回复「在」即部署成功 ✅
