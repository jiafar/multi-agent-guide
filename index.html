5

创建 Discord Bot（每个 Agent 一个）

5.1 创建 Bot 应用

  1. 打开 https://discord.com/developers/applications
  2. 点 New Application → 取名如「选品Agent」
  3. 左侧 Bot → 点 Reset Token → 复制 Token（很重要！）
  4. 开启 Message Content Intent（必须！否则收不到消息内容）
  5. 用 OAuth2 URL 邀请 Bot 进你的 Discord 服务器
⚠️ 注意：每个 Agent 重复上述步骤一次，你会得到 6 个不同的 Bot Token。Token 只显示一次，务必保存好！

5.2 配置多账号

{

    "discord": {
      "enabled": true,
      "accounts": {

          "botName": "主管"
        },
        "xuanpin": {
          "token": "选品bot的token",
          "botName": "选品Agent"
        },
        "dingjia": {
          "token": "定价bot的token",
          "botName": "定价Agent"
        },
        "listing": {

6

设置协作规则（防循环）

最关键的一步——防止 Bot 之间无限循环对话：

{
  "channels": {
    "discord": {
      "allowBots": "mentions",
      "guild": {
        "你的服务器ID": {
          "agents": {
            "zhuguan":  { "requireMention": false },
            "xuanpin":  { "requireMention": true },
            "dingjia":  { "requireMention": true },
            "listing":  { "requireMention": true },
            "tuiguang": { "requireMention": true },
            "kefu":     { "requireMention": true }
          }
        }
      }
    }
  }
}
配置项含义
allowBots: "mentions"Bot 之间只有被 @ 才触发响应
主管 requireMention: false主管看到所有消息，随时可以介入
专员 requireMention: true专员只有被 @ 才回复，避免抢话
