核心概念

从 Chatbot
到 AI Agent

Chatbot 是对话接口——你问，它答，对话结束即遗忘。
AI Agent 是智能代理——它有工具、有记忆、有规划能力，给它一个目标，它自己想办法完成。

💡 本质区别
区别不在模型大小，在于有没有工具调用能力
💬 Chatbot
  • ⬜ 被动响应，你问我答
  • ⬜ 对话结束即遗忘
  • ⬜ 只能输出文字
  • ⬜ 无法使用工具
工具能力
10/100
🤖 AI Agent
  • ✅ 主动规划，目标驱动
  • ✅ 持久记忆，写入文件
  • ✅ 调用API/数据库/文件系统
  • ✅ 自动分解子任务

// ❌ 错误 sessions_spawn(...) // 直接结束 turn // 结果丢失！ // ✅ 正确 sessions_spawn(...) × N sessions_yield() // 等回调 // 汇总展示结果

从零开始

30 分钟搭建你自己的 AI 团队

6个步骤，从安装到验收完成

01
安装 OpenClaw
在 Mac/Linux 安装运行底座
npm install -g openclaw
openclaw init && openclaw gateway start
02
创建 Discord Bot
在 Discord Developer Portal 为每个 Agent 创建独立 Bot，获取 Token，邀请进服务器（需开启 Message Content Intent）
03
配置 openclaw.json
主管设 requireMention: false，专员设 true；主管用 Opus 模型，专员用 Sonnet
04
创建工作空间 + SOUL.md
每个 Agent 独立目录，SOUL.md 定义角色职责、响应规则、工具偏好
05
安装 Skills
openclaw skills install listing-writer
openclaw skills install rufus-geo-optimizer
openclaw skills install linkfoxagent
06
存活测试验证 ✅
在频道发「@主管 测试所有 agent 是否正常工作」
5个Agent 40秒内回复「在」即部署成功
